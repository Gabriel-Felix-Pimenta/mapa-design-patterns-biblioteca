# LibManager - Projeto Exemplo (Skeleton)

Este repositório contém um esqueleto de um projeto Spring Boot demonstrando a aplicação de 3 Design Patterns:
- Factory Method (criacional)
- Facade (estrutural)
- Strategy (comportamental)

Estrutura:
- controller/: Controllers REST
- service/: Serviços de negócio
- repository/: Repositórios (interfaces)
- model/: Entidades de domínio
- dto/: DTOs de entrada/saída
- factory/: ItemFactory
- strategy/: LoanStrategy implementations
- facade/: BorrowFacade

Use este esqueleto como ponto de partida para implementar o trabalho da disciplina.
